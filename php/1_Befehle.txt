Datei vom "newphp"-Verzeichnis im soap Verzeichnis ersetzen:
sudo cp client.php /var/www/html/soap/client.php
sudo cp server.php /var/www/html/soap/server.php

sudo cp fcm_register_user.php /var/www/html/soap/fcm_register_user.php
sudo cp fcm_update_and_send.php /var/www/html/soap/fcm_update_and_send.php

sudo cp clear_fcm_Nutzer.php /var/www/html/soap/clear_fcm_Nutzer.php
sudo cp clear_fcm_Nutzer.php /var/www/html/soap/fcm_connect_db.php


Andere Befehle:
----------------------------------------------------------------------

Permission ändern:
sudo chmod 740 /var/www/html/soap/fcm_update_and_send.php

Befehl um FCM Update Script testweiße aufzurufen:
sudo php /var/www/html/soap/fcm_update_and_send.php

Befehl um die fcm_nutzer Tabelle zu leeren:
sudo php /var/www/html/soap/clear_fcm_Nutzer.php
